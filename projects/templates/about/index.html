Acerca de...

AIOO es un servicio en la nube que permite la interacción de formularios de excel con una base de datos centralizados. El servicio permite la autenticación de los formularios y los usuarios.

El middleware fue desarrollado por Bee888.io con licencia MIT, el servicio no almacena información, solo tranfiere información entre una instancia MongoDB|MySQL y formularios de excel que utilicen el cliente del middleware de Excel o módulos que requieran del router Ajax.
